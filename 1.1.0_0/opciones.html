Frequently asked questions

There is a screen asking for my Gmail's username and password. How can I get rid of it?

Si aparece un cuadro de diálogo pidiendo el nombre de usuario y la contraseña de Gmail, es por causa del livetile de Gmail. La función del livetile de Gmail funciona tomando la información de la sesión que está abierta, y si no hay ninguna, pide las credenciales.
Si desea que deje de aparecer este cuadro, inicie sesión en Gmail o desactive el livetile.

How could I change the language of new metroTab?

The extension works in the language configured on the browser. Currently, the available languages for extension are: english, français, türkçe, español, italiano, magyar, português, русский, nederlands, עברית and 中文 (简体)

Can I create Live Tile by my own?

Right now, users only can create live tiles using RSS feeds. If your mail has a method to get a RSS feed, you can add it easily.
However, more advanced tiles must be programmed into the extension code, because parsing the page is a more complicated process.

If you have a suggestion for a live tile, use the suggestion function in the support page.

There are a lot of cities missing for the weather tile. Any chance to add more locations?

The cities are managed from wunderground.com database. This source was chosen due its forecast reliability. I cannot do anything if your city is not in their lists. Sorry.

I made a RSS live tile, but it doesn't show images. Why?

RSS live tiles searches for images within every post content, from the RSS feed. If the feed only contains summaries without images, the tiles will not have them.

You can force a feed to have the complete post content using a RSS completing service, search them by "full rss". You provide them the original feed URL and they give you another URL with the same feed, but with complete content. Change the feed URL to this and try again.

Is it possible to have a Live Tile which will display the current local weather conditions?

Check if you have the "Weather" tile. You can change the location in the settings (User Icon -> Settings -> Live Tiles -> Weather).

I set a HD image as background. Why new metroTab becomes laggy?

Any site with a HD image as background will be laggy, because the browser must resize it before rendering it.
Use a smaller image or resize it to the screen size.

Is there a way I can change tile's icon to default?

Only if it's a default tile. You can delete it and then restore it from the "Restore tiles" menu.
If tile was created by you, you can change the icon manually or recreate the tile.

Will be configs and color settings saved in Cloud?

Currently no, but in future versions it will be. For now, save your config files on your disk or in Dropbox/Google Drive.

The Chrome's default new tab page has a "Other devices" feature. Is there something like this in new metroTab?

No. In fact, any new tab replacement extension has it. That's because this feature require a Chrome API that currently doesn't exists.

To use the "Other Devices" feature, you can use the “Original new tab page” tile, which let you get to the original Chrome’s new tab page and use it.

In the "World Time" tile settings, some cities have a "DST" flag. How does it work?

DST means Daylight Saving Time. You have to manually turn on/off this flag for every location twice a year.

Unfortunately, there isn't a reliable source for the DST dates of each country to make this automatic. My own country changes the dates every year.

How can I remove the text from the each tile?

Simple, you can change the content for a blank space.

Will you add the Windows 8.1 small tiles?

The appearance is ready, but I have to work in the positioning algorithm. It could take a while.

How to create a dynamic tile?

Recuerda que también puedes usar la API para crear una live tile tú mismo.
